## Як додати новий курс

1. Створити папку `src/content/uk/courses/[course-slug]/`
2. Додати `index.mdx` з метаданими курсу
3. Додати теми `topic-1.mdx`, `topic-2.mdx` тощо
4. Додубавити переклади в `src/messages/en.json` та `ru.json`
5. Оновити цей файл з описом курсу та темами
