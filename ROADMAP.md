## ✅ Виконано

### v0.2.0 — 27.06.2026

#### ✨ Нові функції
- Лекція 2: "Методичний підхід до оцінки ефективності системи управління"
- Інтерактивні MDX-компоненти: `Definition`, `KeyPoint`, `Warning`, `Example`, `Figure`, `Table`, `SelfCheck`
- Візуалізації: схеми процесу, потенціалу управління, організаційного клімату
- Таблиця компетенцій управлінського персоналу
- Формула розрахунку потенціалу управління
- 8 запитань для самоперевірки
- Favicon (синій для браузерів, білий для iOS)

#### 🐛 Виправлення
- Оптимізовано Footer для всіх розмірів екранів
- Виправлено горизонтальну прокрутку на мобільних

#### 🛠️ Технічні зміни
- Централізоване управління версією через `package.json`
- Додано `overflow-x: hidden`

### v0.1.0 — 26.06.2026

#### ✨ Нові функції
- Головна сторінка з Hero Section

- Footer з навігацією та контактами
- Освітній розділ (Education)

- Темна/світла тема (Dark/Light Mode)

- Адаптивний дизайн (mobile-first)
- Навігація між уроками

#### 📚 Контент

- Next.js 16, TypeScript, Tailwind CSS v4, next-intl, MDX
